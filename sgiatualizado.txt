OBJETIVO DO SISTEMA

Desenvolver um sistema web SaaS completo, modular, escalável e multi-tenant, capaz de atender simultaneamente:

Empresas (RH / Departamento Pessoal / Financeiro)

Igrejas (Administração Eclesiástica e Financeira)

Com módulos independentes, porém totalmente integráveis.

REQUISITOS GERAIS

Sistema web responsivo (desktop e mobile)

Arquitetura modular

Multiempresa e multiigreja

Auditoria completa de ações

Conformidade com LGPD

Backup automático

Relatórios gerenciais

Exportação PDF e Excel

Integração com WhatsApp e E-mail

API REST documentada

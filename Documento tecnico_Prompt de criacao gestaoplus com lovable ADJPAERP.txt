🏛️ Prompt de Criação: GestãoPlus - Sistema Integrado de RH e Gestão Eclesiástica

Visão Geral do Projeto
Crie uma aplicação web SaaS chamada GestãoPlus - um sistema integrado de gestão para Departamento Pessoal (DP/RH) e Administração Eclesiástica, projetado para o mercado brasileiro. O sistema deve ser multi-empresa/multi-igreja, com controle de acesso baseado em perfis, conformidade com LGPD e integração com eSocial.

Stack Tecnológico:
React 18+ com TypeScript
Vite como bundler
Tailwind CSS para estilização
shadcn/ui como biblioteca de componentes
Lucide React para ícones
React Router DOM para navegação
React Hook Form + Zod para formulários
Recharts para gráficos
date-fns para manipulação de datas
Lovable Cloud (Supabase) para backend

👥 Módulo Departamento Pessoal (DP/RH)

1. Lista de Funcionários
Filtros:
Busca por nome/CPF/matrícula
Status: Ativo, Férias, Afastado, Desligado
Tipo de contrato: CLT, PJ, Estágio, Temporário
Departamento/Setor
Ações em Lote:

Impressão de Crachás (padrão CR-80: 5,4cm x 8,6cm)
Exportar lista
Tabela com colunas:

Foto, Nome, Cargo, Departamento, Admissão, Status, Ações
2. Formulário de Funcionário (Multi-abas)

Aba 1: Dados Pessoais
Identificação: Nome completo, CPF, RG (número, órgão, UF, emissão), Data nascimento, Sexo, Estado civil, Nacionalidade, Naturalidade (cidade/UF)
Contato: Email, Telefone, Celular
Endereço: CEP (com busca automática), Logradouro, Número, Complemento, Bairro, Cidade, UF
Dados Adicionais: Escolaridade, Raça/Cor (conforme IBGE), Pessoa com deficiência (tipo), Tipo sanguíneo, Nome da mãe, Nome do pai

Aba 2: Contrato de Trabalho
Dados do Contrato: Matrícula, Data admissão, Tipo contrato (CLT/PJ/Estágio/Temporário), Cargo, CBO, Departamento/Setor, Centro de custo, Local de trabalho
Regime: Categoria eSocial, Regime previdenciário, Natureza da atividade, Indicador de admissão
Sindical: Sindicato, Contribuição sindical
Término: Data término (contratos temporários), Motivo desligamento

Aba 3: Jornada de Trabalho
Horário: Tipo jornada (fixa/flexível/escala), Horas semanais, Entrada, Saída, Intervalo
Escala: Dias da semana trabalhados
Configurações: Permite hora extra, Adicional noturno, Periculosidade, Insalubridade (grau)

Aba 4: Banco de Horas
Configuração: Banco de horas ativo (sim/não), Saldo atual, Data início
Histórico: Tabela com data, tipo (crédito/débito), horas, motivo, saldo resultante
Lançamentos: Formulário para adicionar créditos/débitos

Aba 5: Documentos
CTPS: Número, Série, UF, Data emissão
PIS/PASEP: Número, Data cadastro
CNH: Número, Categoria, Validade
Título de Eleitor: Número, Zona, Seção
Certificado Reservista: Número, Categoria
Upload de Documentos: Área para anexar PDFs/imagens

Aba 6: Benefícios
Vale Transporte: Ativo, Valor diário, Linhas utilizadas
Vale Alimentação/Refeição: Tipo, Valor mensal, Desconto em folha
Plano de Saúde: Operadora, Tipo plano, Valor, Dependentes inclusos
Plano Odontológico: Operadora, Valor
Seguro de Vida: Valor cobertura, Beneficiários
Outros: Auxílio creche, Auxílio educação, Gympass

Aba 7: Dados eSocial
Classificação: Categoria trabalhador, Regime previdenciário, Natureza atividade
Vínculos Anteriores: Histórico de empresas anteriores
Observações: Campo livre para anotações eSocial

Aba 8: Dependentes
Para cada dependente:
Nome completo, CPF, Data nascimento, Grau de parentesco
Deduz IRRF (sim/não)
Salário família (sim/não)
Plano de saúde (sim/não)
Possui deficiência (tipo)

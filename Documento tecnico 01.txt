# DOCUMENTAÇÃO TÉCNICA COMPLETA - ADJPA ERP

## VISÃO GERAL DO SISTEMA

O ADJPA ERP é um sistema completo de gestão empresarial desenvolvido especificamente para igrejas e organizações religiosas. O sistema integra múltiplos módulos para gerenciar todos os aspectos operacionais, financeiros e administrativos da organização.

### TECNOLOGIAS UTILIZADAS

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS
- **Ícones**: Lucide React
- **PDF Generation**: jsPDF + html2canvas
- **Build Tool**: Vite
- **Database**: PostgreSQL (via serviços)
- **Storage**: LocalStorage (dados locais)
